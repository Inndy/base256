# base256 - BIP-39 Word Encoding

Encode arbitrary bytes as human-readable words using a curated subset of the BIP-39 wordlist.

256 words are selected from the original 2048 BIP-39 words to maximize pairwise edit distance, with preference for diverse starting letters and shorter words. Each byte maps to one word.

## Usage

### Python

```python
from base256 import encode, decode

encoded = encode(b"hello")   # "hospital ranch outdoor pebble photo"
decoded = decode(encoded)     # b"hello"
```

CLI:

```sh
echo -n "hello" | python3 base256.py encode
echo "gossip giggle harvest harvest hip" | python3 base256.py decode
```

### Go

```go
import "your/module/gen"

encoded := base256.Encode([]byte("hello"))
decoded, err := base256.Decode(encoded)
```

## Word Selection

`pick_words.py` selects 256 words from `bip39wordlist.txt` using a greedy algorithm that:

- Maximizes minimum pairwise edit distance (weighted)
- Rewards different first characters (bonus weight) for uniform letter distribution
- Prefers shorter words as a tiebreaker
- Enforces a minimum raw edit distance of 2 between any pair

## Code Generation

`scripts/codegen.py` reads `selected_256.txt` and generates `base256.go` and `base256.py`.

```sh
make
```

## File Structure

- `bip39wordlist.txt` - Original BIP-39 2048 wordlist
- `scripts/pick_words.py` - Word selection algorithm
- `scripts/codegen.py` - Generates encoder/decoder source files
- `selected_256.txt` - Selected 256 words
- `base256.go` - Generated Go encoder/decoder
- `base256.py` - Generated Python encoder/decoder
- `base256_test.go` - Go tests
